Handler Types

Hibiki HTML supports 3 types of handlers: inline, remote, and JavaScript. All 3 handler types are isomorphic, they each return a set of actions that are executed by the Hibiki HTML runtime to update the data model.

Anything you can do or change in an inline handler can also be done with a remote handler by returning actions in JSON form.

Inline Handlers

Let's create a button, and add an inline click handler (other supported events include change, mount, and submit). All inline handlers in your HTML end with .handler. 

<button click.handler="$.color = 'red'" class="button">Set Red</button>

Handlers can update data, make backend API calls, call other handlers, and more. This example shows a click handler updating $.color in the data model.
