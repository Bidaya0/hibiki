Getting Started

You can create a Hibiki HTML app on any page in two steps. First add the Hibiki HTML JavaScript tag to your page: 

<script src="https://cdn.hibikihtml.com/hibiki/v0.3.4/hibiki-prod.min.js"></script>

Next add a Hibiki template block to your page where you want the application to display (you can also wrap your existing HTML to instantly convert it to a Hibiki app): 

<template hibiki>
  HTML content goes here
</template>
Hibiki HTML is HTML

Hibiki renders regular HTML as-is. It is parsed using the browser's own parser, so it follows the same rules.

About This Tutorial

The Tutorial is Interactive

You can edit the code in the upper right, hit submit and it will be executed in the bottom right. The examples can also be run in a static HTML file on your own machine, or in Codepen.

How the preview and tutorial was built

The entire tutorial site is written in Hibiki HTML! It includes the excellent CodeMirror library to render the code editor. The full source code is available on GitHub.

The preview you see in the bottom right is an <iframe> that renders the HTML in the code block above. Here is the full template (GitHub). It imports the Hibiki HTML script tag, and for styling I've imported the excellent Bulma CSS library, and FontAwesome v4.7. 

<script src="https://cdn.hibikihtml.com/hibiki/v0.3.4/hibiki-prod.min.js"></script>
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.9.3/css/bulma.min.css">
