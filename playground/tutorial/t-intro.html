Hibiki Introduction

Hibiki HTML is a powerful new frontend web framework that lets you create modern, dynamic, frontend applications without JavaScript, that can be fully scripted and controlled by backend code. Key features:

  • Easy to Get Started
  • No JavaScript Stack to Setup
  • Backend Control
  • No Scaffolding
  • Libraries and Components
  • Open Source
  • Support & Help

Easy to Get Started

Add one script tag to the top of your page:
https://cdn.hibikihtml.com/hibiki/v0.3.2/hibiki-prod.min.js

Wrap any portion of your content with a <template hibiki> tag and you have your first Hibiki HTML app. All plain HTML content is rendered as is, and because Hibiki HTML uses the browser's HTML parser, it follows the same rules as standard HTML.

Hibiki HTML is compatible with any backend language or framework, any CSS library, and any existing backend template language that you may be currently using.

No JavaScript Stack to Setup

JavaScript is great, but not great for every project. If you have a backend in Python, Java, Golang, Rust, or C#, adding a complete JavaScript eco-system (npm, webpack, babel, React, redux, etc.) to get frontend functionality is overkill.

Hibiki HTML handlers easily integrate with your backend REST APIs, sending and receiving JSON data.

Backend Control

By returning Hibiki Actions from your backend's JSON handlers you can fully script any frontend action.

Hibiki HTML actions are isomorphic, meaning that any action that can be triggered on the frontend can also be triggered from the backend using special JSON return values. This lets you write complex frontend logic or validation in your prefered backend language, using the libraries and functions you are already familiar with.

No Scaffolding

You can write a full app in one HTML file. No complex build tools to learn, configuration files, deployment, or server setup. Hibiki HTML can be embedded into scripts, written as plain HTML to the file system, or used to add dynamic functionality to Jamstack sites, Wikis, or internal applications.

Libraries and Components

Hibiki HTML makes it easy to use, share, and bundle reuseable components. The ecosystem is just getting started but the component system allows for linking 3rd party JavaScript libraries (D3, CodeMirror, etc.) and importing ReactJS components.

Open Source

The only restriction is if you modify any of the *Hibiki HTML source files* you must make the source code of those changes available.
